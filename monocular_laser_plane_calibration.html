单目光平面标定系统

Monocular Laser Plane Calibration System

标定原理

基本原理

通过单个相机观察激光平面与标定板的交线，确定激光平面在相机坐标系中的精确位置和方向。

关键步骤

  1. 相机内参标定
  2. 标定板位姿估计(PnP)
  3. 激光条纹提取
  4. 三维坐标计算
  5. 光平面拟合
光平面方程: Ax + By + Cz + D = 0
投影方程: s[u,v,1]ᵀ = K[R|t][X,Y,Z,1]ᵀ
